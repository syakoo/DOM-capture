My Chrome Extension

## Usage

1. Clone this repo
```bash
$ git clone <this repository url>
```

2. Install dependecies
```bash
$ npm install
```

3. Build it
```bash
$ npm run build
```

4. Go to `chrome://extensions/` and activate `Developer Mode`

5. Click `Load unpacked`

6. Find and select the build dir of the app.
